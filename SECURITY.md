## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Email:** _(maintainer: add your preferred contact)_

Please do **not** open a public issue for security vulnerabilities.

You should expect an initial response within 72 hours. Once the issue is confirmed, a fix will be prioritized and released as soon as possible.
